# Anime Telegram Bot (Heroku-Optimized)

## Features
- Monitors multiple anime on SubsPlease.
- Downloads new 1080p episodes automatically.
- Encodes to 720p.
- Posts directly to Telegram channel.
- Cleans up files to save Heroku storage.
